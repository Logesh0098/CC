Amazon Elastic Compute Cloud (EC2)

üìò Concept Introduction

Amazon Elastic Compute Cloud (EC2) is a foundational web service from Amazon Web Services (AWS) that provides secure, resizable compute capacity in the cloud. As a core Infrastructure as a Service (IaaS) offering, it is designed to make web-scale cloud computing easier for developers by providing virtual servers, known as "instances," that can be provisioned in minutes.

üß© Analogy: Renting a Car

Using EC2 is like renting a car. Instead of buying, insuring, and maintaining your own vehicle, you go to a rental agency. You can choose the exact model you need (an economy car for city driving, an SUV for a family trip, a sports car for fun), pay for it only for the time you use it (by the hour or day), and simply return it when you're done. EC2 lets you do the same with servers, eliminating the massive upfront cost and long-term commitment of physical hardware.

üîç In-Depth Explanation

Core Components of EC2

  • Instances: These are the virtual servers. They come in a wide variety of types, optimized for different tasks.
  • Amazon Machine Images (AMIs): Pre-configured templates for your instances that package the OS and other software. You can choose from standard AMIs (e.g., Ubuntu, Windows Server) or create your own.
  • Instance Types: A family of instances optimized for different workloads, such as General Purpose (M, T series), Compute Optimized (C series), Memory Optimized (R, X series), and Accelerated Computing (P, G, Trn series for ML/graphics).
  • Elastic Block Store (EBS): Persistent block storage volumes for use with EC2 instances, similar to a hard drive for your virtual server.
  • Security Groups: A virtual firewall for your instances to control inbound and outbound traffic. It's a fundamental security layer.
  • Pricing Models: EC2 offers flexible pricing: On-Demand (pay by the second), Spot Instances (bid on unused capacity for huge savings, but can be interrupted), Reserved Instances (commit to 1-3 years for a large discount), and Savings Plans (commit to a certain spend for a discount).

üß† Mini Quiz

1. What is an Amazon Machine Image (AMI)?

An AMI is a pre-configured template used to create an EC2 instance. It includes the operating system, application server, and applications, acting as a blueprint for the virtual server's root volume.

2. Which EC2 pricing model offers the largest discount but carries the risk of termination?

Spot Instances. They let you take advantage of unused EC2 capacity at discounts of up to 90% off the On-Demand price, but AWS can reclaim the capacity with a two-minute warning.

3. What AWS component acts as a virtual firewall for an EC2 instance?

A Security Group. It controls inbound and outbound traffic at the instance level, specifying which protocols, ports, and source/destination IP ranges are allowed.
